Curso Git

Comandos básicos:

  • git init
  • git status -s
  • git add
  • git add .
  • git commit -m "msj"
  • git commit -am "msj"
  • git log --oneline
  • git reset --hard [hash]
  • git commit --amend

Luego de git remote add origin, ejecutar git push origin master para logearse y poder subir el código a Github
